# Portfolio — Helen Aguertin

Ce dépôt contient votre portfolio web (fichiers statiques).

Aperçu rapide
- Ouvrez `index2.html` dans un navigateur pour prévisualiser le site localement.

Préparation Git
- Si `git` est installé, exécutez le script `push_and_init.ps1` (PowerShell) depuis la racine du projet pour initialiser le dépôt, créer le commit initial et pousser vers GitHub.
- Voir `README_AUTH.md` pour l'installation de Git et les options d'authentification (PAT ou SSH).
